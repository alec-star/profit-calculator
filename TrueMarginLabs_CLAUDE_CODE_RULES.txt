=====================

Before Claude Code ships any feature:

1. LINT CHECK
   - Run linter (ESLint/Stylelint)
   - Fix all warnings and errors
   - Check for unused code

2. FUNCTIONALITY TEST
   - All hover states work (cards, buttons, rows)
   - All clicks/interactions respond
   - Mobile responsive (test at 375px, 768px, 1440px)
   - Colors match hex codes (#00d084 emerald, #ffffff white, #0a0a0a black)
   - Text contrast is readable (WCAG AA minimum)
   - Form inputs work correctly
   - Email validation works

3. BRAND COMPLIANCE CHECK
   - [ ] White backgrounds only (no colored backgrounds)
   - [ ] Emerald green only (no orange, yellow, red, blue)
   - [ ] System fonts only (no web fonts)
   - [ ] Emojis instead of custom icons
   - [ ] Bold headlines (weight 800)
